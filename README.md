# SweetCrumbs Bakery

A multi-section landing page for **SweetCrumbs Bakery**, a fictional artisan bakery based in Nairobi, Kenya. This project was built as a Web Design Phase One assignment focused on traditional, pre-CSS HTML techniques.

## Purpose

The goal of this project is to demonstrate core HTML capabilities — page structure, tables for layout, native form controls, and internal anchor navigation — using **only HTML**, with no CSS (inline, internal, or external) and no JavaScript.

## Live File

The complete site lives in [`SweetcrumbsBakery.index.html`](./SweetcrumbsBakery.index.html). Open it directly in any browser — no build step, server, or dependencies required.

## Sections

The page is a single scrolling document with internal anchor navigation between:

- **Top utility bar** — address, phone, opening hours
- **Header** — bakery name and tagline
- **Navigation bar** — Home, About, Menu, Specials, Gallery, Contact
- **Hero banner** — welcome message and two call-to-action buttons
- **Announcement strip** — weekly promo code
- **About Us** — bakery history and key stats (years in business, menu size, customers served)
- **Menu** — 8 bakery items with description, price, and availability
- **Weekly Specials** — Monday, Wednesday, and Friday offers
- **Gallery** — emoji-based visual showcase of products
- **Customer Testimonials** — three customer reviews with star ratings
- **Contact** — contact details and an inquiry form (name, email, phone, inquiry type, message)
- **Location** — address, directions, and a Google Maps link
- **Footer** — quick links, popular products, business hours, and policy links

## Techniques Used

- `<table>`-based page and content layout throughout
- Presentational HTML attributes: `bgcolor`, `align`, `width`, `cellpadding`, `cellspacing`, `<font color size>`
- Native HTML form controls: `input`, `select`, `textarea`, submit/reset buttons
- Internal anchor navigation (`#home`, `#about`, `#menu`, `#specials`, `#gallery`, `#contact`)
- `<marquee>` for lightweight navigation and headline animation — a native HTML element that requires no CSS or JavaScript

## Constraints

Per the assignment, this project deliberately avoids:

- CSS (inline, internal, or external)
- JavaScript
- SVG, Canvas, or external UI frameworks
- Flexbox or Grid layout
